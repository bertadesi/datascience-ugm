transforming unstructured personal email data into Financial Expense Categorization
This project is initial project for experimenting how the subject objective can be done

How to run
run email_rawdataset_Extraction.py --> for preparing raw dataset
run test_panda_processing.py --> for building a ML model
run demo-code-email-ML.py --> for extracting a ML structured data
run demo-code-email.py --> for extracting a manual structured data
run pull_report.py --> for visualizing the financial expense categorization

## Notes
To use the above code need to download token from https://console.cloud.google.com/apis/credentials
also please make sure all needed libraries already being installed
